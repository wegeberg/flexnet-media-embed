Flexnet Media Embed

A plugin for adding media content to TinyMCE 4. Add Facebook, Twitter, YouTube, Vimeo, Soundcloud, Infogram

Dependant on jQuery and PHP.

  • Place the cursor in the textarea where you want the media embedded.
  • Click on the plugin button
  • Insert the URL for the embed and press OK.
    E.g.: https://vimeo.com/8870338 | https://www.facebook.com/uniladmag/videos/2530516633638047/
